# Big Data + ML Pipeline Summary

This folder demonstrates a free-safe Big Data and ML-style pipeline for the CloudOps project.

## Files

- `cloudops_metrics.csv` contains sample cloud monitoring metrics.
- `ml_pipeline.py` reads the dataset, analyzes CPU usage, request count, and critical events.
- The script gives a simple prediction about system load.

## Purpose

This module represents the Big Data and ML requirement without using paid services like EMR, Redshift, Kinesis, or SageMaker.

## Output

The script calculates:
- Average CPU usage
- Average request count
- Critical events
- Load prediction
