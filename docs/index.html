CloudOps Enterprise Platform

No-Code / Low-Code Frontend Prototype

This page represents the no-code cloud application layer of the project.

Cloud Application

Flask web application deployed on AWS EC2 using Docker.

Serverless Backend

AWS Lambda Function URL is used as a serverless API trigger.

Cloud Data Layer

Amazon S3 provides storage and DynamoDB provides database support.

Monitoring & Security

CloudWatch, IAM role, Security Groups, and AWS Budget alert are used.
